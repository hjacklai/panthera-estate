# pantheraestate.com

Marketing site for **Panthera Bertam**, a development of 3 bungalows and 16 semi-detached homes around a shared central park in Bertam, Penang.

Static site (HTML / CSS / JS, no build step). Deployed automatically via GitHub Pages on the custom domain `pantheraestate.com`. Push to `main` and it redeploys.

Developed by Panthera Development Sdn. Bhd.
